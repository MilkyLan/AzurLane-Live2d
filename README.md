### 各文件夹内容示意

前两个文件夹为`/Android/data/`下的源文件,其余文件夹是解包后的资源

| 文件夹名              | 内容               |
| --------------------- | ------------------ |
| com.bilibili.azurlane | 国服游戏源文件     |
| com.YoStarJP.AzurLane | 日服游戏源文件     |
| live2d                | 角色live2d皮肤资源 |
| loadingbg             | loading界面背景图  |
| painting              | 角色立绘图         |
| spinepainting         | 角色ANI皮肤资源    |
